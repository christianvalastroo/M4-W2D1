# 📸 Pexels Gallery

Una semplice web app che permette di cercare immagini utilizzando l'API di **Pexels** e mostrarle in una galleria.

L'utente può scrivere una parola nella barra di ricerca e visualizzare le immagini correlate.

---

## 🚀 Tecnologie utilizzate

- HTML5
- CSS3
- Bootstrap 5
- JavaScript
- Fetch API
- Pexels API

---

## ⚙️ Funzionamento

1. L’utente inserisce una parola nella barra di ricerca.
2. JavaScript invia una richiesta all’API di Pexels.
3. L’API restituisce le immagini.
4. JavaScript crea dinamicamente delle **card** con le immagini.
5. Le card vengono mostrate nella pagina.

---

## 🧠 Concetti JavaScript utilizzati

- async / await
- fetch()
- DOM manipulation
- createElement()
- appendChild()
- forEach()

---

## 🎨 Design

L'interfaccia è realizzata con **Bootstrap** e **CSS personalizzato** per creare una galleria moderna e responsive.
